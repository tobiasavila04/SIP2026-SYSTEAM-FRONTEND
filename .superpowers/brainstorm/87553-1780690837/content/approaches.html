¿Cómo organizamos la página /ganancias?

Tres enfoques distintos — cada uno tiene implicaciones en UX y complejidad.

A

Tabs por rol Recomendado

Una pestaña "Mis Proyectos" (creator) y otra "Mis Inversiones" (investor). Tabs visibles según el rol. Si tenés ambos roles, ves ambas tabs.

Pros

  • Separación clara por rol
  • Cada sección ocupa todo el ancho
  • Escala bien con mucho contenido

Contras

  • Un click extra para cambiar de vista
B

Secciones apiladas

Todo en una página scrollable: primero la sección de creator, después la de investor. Ambas visibles al mismo tiempo si el usuario tiene ambos roles.

  • Todo visible de un vistazo
  • Sin clicks de navegación

Contras

  • Página larga si hay mucha data
  • Mezcla contextos distintos
C

Lista + Panel de detalle

Columna izquierda con los proyectos/inversiones. Click en uno → panel derecho muestra el detalle completo, repartos e historial.
